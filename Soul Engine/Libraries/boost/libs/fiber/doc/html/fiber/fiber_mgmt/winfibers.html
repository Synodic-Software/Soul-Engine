Using WinFiber-API

Because the TIB (thread information block) is not fully described in the MSDN, it might be possible that not all required TIB-parts are swapped. With compiler flag BOOST_USE_WINFIBERS fiber uses internally the Windows Fiber API (see also Boost.Context WinFiber support).
